# Text-to-SQL System: Tehqeeq.AI

A basic Text-to-SQL system that converts natural language queries into SQL queries and returns structured, formatted results from a relational database.

---

## 🚀 Overview

This project allows users to interact with a database using natural language instead of writing SQL manually. It processes user queries, generates SQL statements, executes them on a structured database, and returns readable results.

Initially, the focus was on query generation, but a major challenge was designing and normalizing a 3500+ row CSV dataset into a proper relational schema with multiple tables.

---

## ✨ Features

- Natural language to SQL conversion (basic version)
- Executes SQL queries on structured relational database
- Returns formatted query results
- Works with normalized multi-table schema
- Simple and lightweight pipeline

---

## 🧠 Key Learning

- Data cleaning and schema design is harder than query generation
- Importance of normalization in real-world datasets
- Understanding relationship mapping between tables
- Basic NLP-to-database interaction pipeline

---

## 🔧 Tech Stack

- Python
- SQL (MySQL / SQLite)
